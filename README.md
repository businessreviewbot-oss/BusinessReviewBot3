# BusinessReviewBot\n\nAI-powered review response automation platform built with Next.js, NextAuth, OpenAI, and Stripe.\n\n## Features\n\n- **AI-Powered Responses**: Generate professional, personalized review responses using GPT-4o-mini\n- **Google Authentication**: Secure sign-in with NextAuth and Google OAuth\n- **Stripe Integration**: Subscription billing with monthly and annual plans\n- **Modern UI**: Beautiful dark theme with glassmorphism effects and Framer Motion animations\n- **Responsive Design**: Fully responsive across all devices\n- **Multiple Sections**: Hero, Magic Response Generator, Features, Pricing, FAQ, and Footer\n\n## Tech Stack\n\n- **Framework**: Next.js 16 (App Router)\n- **Styling**: Tailwind CSS 4\n- **Animations**: Framer Motion\n- **Icons**: Lucide React\n- **Authentication**: NextAuth.js with Google Provider\n- **AI**: OpenAI API (GPT-4o-mini)\n- **Payments**: Stripe\n- **Language**: TypeScript\n\n## Project Structure\n\n```\napp/\n├── api/\n│   ├── auth/[...nextauth]/route.ts  # NextAuth API route\n│   ├── magic-response/route.ts      # AI response generation\n│   └── checkout/route.ts            # Stripe checkout session\n├── components/\n│   ├── Navbar.tsx                   # Navigation with auth\n│   ├── HeroSection.tsx              # Landing hero\n│   ├── MagicResponseGenerator.tsx   # Interactive AI demo\n│   ├── FeaturesSection.tsx          # Feature grid\n│   ├── PricingSection.tsx           # Pricing cards with Stripe\n│   ├── FAQSection.tsx               # Accordion FAQ\n│   ├── Footer.tsx                   # Footer with social links\n│   └── Button.tsx                   # Reusable button component\n├── success/\n│   └── page.tsx                     # Post-purchase success page\n├── layout.tsx                       # Root layout with Providers\n├── page.tsx                         # Main landing page\n├── providers.tsx                    # NextAuth SessionProvider\n└── globals.css                      # Global styles\nlib/\n└── auth.ts                          # NextAuth configuration\n```\n\n## Getting Started\n\n### 1. Clone and Install\n\n```bash\nnpm install\n```\n\n### 2. Environment Variables\n\nCreate a `.env.local` file in the root directory:\n\n```env\n# NextAuth Configuration\nNEXTAUTH_URL=http://localhost:3000\nNEXTAUTH_SECRET=your-secret-key-here-change-in-production\n\n# Google OAuth\nGOOGLE_CLIENT_ID=your-google-client-id\nGOOGLE_CLIENT_SECRET=your-google-client-secret\n\n# OpenAI API\nOPENAI_API_KEY=your-openai-api-key\n\n# Stripe\nSTRIPE_SECRET_KEY=your-stripe-secret-key\nNEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your-stripe-publishable-key\nSTRIPE_PRICE_ID_MONTHLY=your-monthly-price-id\nSTRIPE_PRICE_ID_ANNUAL=your-annual-price-id\n```\n\n### 3. Set Up Google OAuth\n\n1. Go to [Google Cloud Console](https://console.cloud.google.com/)\n2. Create a new project or select existing one\n3. Enable Google+ API\n4. Create OAuth 2.0 credentials\n5. Add authorized redirect URI: `http://localhost:3000/api/auth/callback/google`\n6. Copy Client ID and Client Secret to `.env.local`\n\n### 4. Set Up OpenAI\n\n1. Get API key from [OpenAI Platform](https://platform.openai.com/)\n2. Add to `.env.local`\n\n### 5. Set Up Stripe\n\n1. Get keys from [Stripe Dashboard](https://dashboard.stripe.com/)\n2. Create products and prices for monthly/annual plans\n3. Add price IDs to `.env.local`\n\n### 6. Generate NextAuth Secret\n\n```bash\nopenssl rand -base64 32\n```\n\nAdd the generated secret to `.env.local` as `NEXTAUTH_SECRET`\n\n### 7. Run Development Server\n\n```bash\nnpm run dev\n```\n\nOpen [http://localhost:3000](http://localhost:3000) in your browser.\n\n## API Routes\n\n### POST /api/magic-response\n\nGenerates AI-powered review responses.\n\n**Request:**\n```json\n{\n  \"review\": \"Great service! Loved the experience.\"\n}\n```\n\n**Response:**\n```json\n{\n  \"response\": \"Thank you so much for your kind words! We're thrilled you had a great experience.\"\n}\n```\n\n### POST /api/checkout\n\nCreates a Stripe checkout session.\n\n**Request:**\n```json\n{\n  \"priceId\": \"price_xxx\",\n  \"mode\": \"subscription\"\n}\n```\n\n**Response:**\n```json\n{\n  \"url\": \"https://checkout.stripe.com/...\"\n}\n```\n\n## Pages\n\n- **/** - Landing page with all sections\n- **/success** - Post-checkout success page\n\n## Components Overview\n\n### Navbar\n- Sticky header with authentication state\n- Sign in/out buttons\n- Responsive design\n\n### HeroSection\n- Eye-catching headline with gradient text\n- CTA buttons for trial signup\n- Trust indicators\n\n### MagicResponseGenerator\n- Interactive demo of AI functionality\n- Copy-to-clipboard feature\n- Real-time response generation\n\n### FeaturesSection\n- Grid of 8 key features\n- Icon-based cards with hover effects\n- Glassmorphism design\n\n### PricingSection\n- Three pricing tiers\n- Monthly/Annual toggle with savings indicator\n- Stripe checkout integration\n- Feature comparison\n\n### FAQSection\n- Accordion-style FAQ\n- Smooth animations\n- 8 common questions covered\n\n### Footer\n- Brand information\n- Social media links (Twitter, LinkedIn, GitHub)\n- Navigation links\n- Copyright notice\n\n## Design System\n\n### Colors\n- Primary Background: `#06162c` (dark blue)\n- Accent Blue: `#00c2ff`\n- Accent Purple: `#8b5cf6`\n- Text: White with various opacity levels\n\n### Effects\n- Glassmorphism: `backdrop-blur-xl` with transparent backgrounds\n- Gradients: Blue to purple accents\n- Shadows: Colored shadows for depth\n- Animations: Framer Motion for smooth transitions\n\n## Deployment\n\n### Vercel (Recommended)\n\n1.
