# Quick Setup Guide for BusinessReviewBot\n\nThis guide will help you get BusinessReviewBot up and running in under 10 minutes.\n\n## Prerequisites\n\n- Node.js 18+ installed\n- npm or yarn\n- Google Cloud account (for OAuth)\n- OpenAI API account\n- Stripe account\n\n## Step-by-Step Setup\n\n### 1. Install Dependencies\n\n```bash\nnpm install\n```\n\n### 2. Configure Environment Variables\n\nCopy the `.env.local` file and fill in your credentials:\n\n```bash\ncp .env.local .env.local.backup  # Keep a backup\n```\n\nEdit `.env.local` with your actual values:\n\n#### NextAuth Secret (Required)\nGenerate a secure secret:\n```bash\nopenssl rand -base64 32\n```\nAdd to `.env.local` as `NEXTAUTH_SECRET`\n\n#### Google OAuth (Required for Sign In)\n1. Go to https://console.cloud.google.com/\n2. Create a new project: \"BusinessReviewBot\"\n3. Navigate to \"APIs & Services\" > \"Credentials\"\n4. Click \"Create Credentials\" > \"OAuth 2.0 Client ID\"\n5. Configure consent screen if prompted\n6. Application type: \"Web application\"\n7. Authorized redirect URIs: `http://localhost:3000/api/auth/callback/google`\n8. Copy Client ID and Client Secret to `.env.local`\n\n#### OpenAI API (Required for AI Responses)\n1. Go to https://platform.openai.com/api-keys\n2. Create new API key\n3. Copy to `.env.local` as `OPENAI_API_KEY`\n\n#### Stripe (Required for Payments)\n1. Go to https://dashboard.stripe.com/\n2. Get your test API keys from Developers > API keys\n3. Copy Secret key to `.env.local` as `STRIPE_SECRET_KEY`\n4. Copy Publishable key to `.env.local` as `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`\n\n**Create Products:**\n1. Products > Add Product\n2. Create \"BusinessReviewBot Monthly\" - $29/month recurring\n3. Create \"BusinessReviewBot Annual\" - $299/year recurring\n4. Copy the Price IDs to `.env.local`\n\n### 3. Start Development Server\n\n```bash\nnpm run dev\n```\n\nVisit http://localhost:3000\n\n### 4. Test the Application\n\n#### Test Authentication:\n1. Click \"Sign In\" in navbar\n2. Sign in with Google\n3. Verify you see \"Sign Out\" button and your email\n\n#### Test AI Response Generator:\n1. Scroll to \"Magic Response Generator\" section\n2. Paste a sample review: \"Great service! Very helpful staff.\"\n3. Click \"Generate Response\"\n4. Verify AI generates a professional response\n\n#### Test Stripe Checkout:\n1. Scroll to \"Pricing\" section\n2. Click \"Get Started\" on any plan\n3. Verify Stripe checkout page opens\n4. Use test card: 4242 4242 4242 4242\n5. Any future expiry date, any CVC\n6. Complete checkout\n7. Verify redirect to /success page\n\n## Common Issues\n\n### \"Invalid redirect URI\" error\n- Double-check the redirect URI in Google Console matches exactly\n- Make sure there's no trailing slash\n- Format: `http://localhost:3000/api/auth/callback/google`\n\n### \"OpenAI API Error\"\n- Verify API key is correct\n- Check you have credits in your OpenAI account\n- Ensure no extra spaces in the .env.local file\n\n### \"Stripe checkout not opening\"\n- Verify price IDs are correct\n- Check Stripe API keys are for test mode\n- Look for errors in browser console\n\n### Port 3000 already in use\n```bash\n# Kill process on port 3000\nlsof -ti:3000 | xargs kill -9\n# Then run npm run dev again\n```\n\n## Production Deployment Checklist\n\nBefore deploying to production:\n\n1. [ ] Update `NEXTAUTH_URL` to your production domain\n2. [ ] Generate new `NEXTAUTH_SECRET` for production\n3. [ ] Update Google OAuth redirect URI to production URL\n4. [ ] Switch Stripe to production mode and update keys\n5. [ ] Update OpenAI API key if using different account\n6. [ ] Set up Stripe webhooks for production\n7. [ ] Test all flows in production environment\n\n## Need Help?\n\n- Check the main README.md for detailed documentation\n- Review API route files for implementation details\n- Check Next.js console for error messages\n- Verify all environment variables are set correctly\n\n## Quick Test Commands\n\n```bash\n# Verify Node version\nnode --version  # Should be 18+\n\n# Check if server is running\ncurl http://localhost:3000\n\n# View logs\n# Check terminal running npm run dev\n\n# Build for production (test before deploying)\nnpm run build\n```\n\n## Success! ðŸŽ‰\n\nIf you can:\n- âœ… Sign in with Google\n- âœ… Generate AI responses\n- âœ… See Stripe checkout page\n- âœ… Navigate to success page after checkout\n\nYou're all set!
